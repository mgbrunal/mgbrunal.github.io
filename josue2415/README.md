# J24:15 — Software House · Brand assets

Legal entity: **JOSUE 24:15 SAS**

## Concept

The J carries an integrated Latin cross (top arm above the crossbar, equal left/right arms, J hook below) inside a soft chimney-house silhouette. The numerals 24:15 stack tightly to the right of the J, anchored to the same vertical line — a direct reference to Joshua 24:15.
